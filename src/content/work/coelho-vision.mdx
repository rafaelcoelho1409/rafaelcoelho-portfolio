## Four CV pipelines, one app

COELHO VISION is a Computer Vision platform built on OpenCV, Ultralytics, and MediaPipe.

1. **Object Detection** — object / image classification + face detection
2. **Image Segmentation** — pixel-level segmentation for detailed analysis
3. **Pose Estimation** — gesture recognition and motion analysis
4. **Live Camera Integration** — apply all three in real time from the user's device camera

Used as a sandbox for evaluating tradeoffs between YOLO variants and MediaPipe across realtime
and high-accuracy scenarios.
