## Personal portfolio meets financial ML

COELHO Finance was built as a personal portfolio project to test **Data Science, Machine Learning,
and AI tools applied to financial markets**. It helps users make informed investment decisions,
track holdings, and identify opportunities — all in a free, live Streamlit app.

The project doubled as a deliberate sandbox: every new ML framework I wanted to learn got
prototyped against real-market data here before going into production work.
